# ClipMaker

NLW Operator - Iniciante


Sobre o projeto

O ClipMaker é uma aplicação web que demonstra um fluxo de upload de vídeo com Cloudinary e identificação de "momento viral" via API integrado ao GEMINI IA.


Tecnologias

HTML5
Tailwind CSS
JavaScript (ES6+)
Gemini API
Cloudinary


Objetivos de aprendizagem

Praticar organização de um projeto front-end,
Manipulação do DOM com JavaScript,
Consumo de APIs externas,
Fluxo básico de upload de arquivos.


Como executar

Abra o arquivo index.html no navegador (ou use uma extensão como Live Server no VS Code).

Insira a sua API Key do Gemini no campo apropriado para que a função de exemplo funcione (a API key é usada apenas como exemplo no código).
Clique em "Começar Upload" para abrir o widget de upload do Cloudinary.

Observação: este projeto é um exemplo; para uso real você deve revisar e ajustar as chaves, credenciais e endpoints, e tratar dados sensíveis adequadamente.
